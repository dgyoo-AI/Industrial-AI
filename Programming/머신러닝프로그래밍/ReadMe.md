### Bird Image 학습 및 인식 프로그램.

학습 데이터는 Caltech 에서 제공하는 Bird Image 로 총 200 품종이며 약 23,000 개의 Image 파일이다.   
학습하여 결과를 출력하기까지 많은 시간 및 컴퓨터 성능이 필요한 관계로 각 Class 단위, Image 갯 수 단위로   
학습하여 학습한 데이터 증가에 따른 인식률의 상관관계를 확인 하였다.

학습에 필요한 Image 는 용량이 큰 관계로 다음 경로에서 Download 가능하다.
